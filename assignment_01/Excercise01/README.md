Exercise – Printing a Triangle

Problem 1
Input N ( number of lines , for example 6 ) and character to print, from keyboard;
Write a function func1() that outputs a triangle of height and width N

Example:

Input:
N = 6
Character = *

Output:
*
**
***
****
*****
******
